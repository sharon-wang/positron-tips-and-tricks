What is Positron Assistant?
[MELISSA]

Positron Assistant is an AI client that provides LLM integration with Positron through chat and inline completions. 

It‚Äôs great for help with debugging, refactoring, or getting suggestions for next steps in your projects. 

Two concepts to understand to make the best use of Assistant are context and tools.

36

Context beyond files on disk
Aware of active console/runtime
Aware of structure of in-memory objects
Ask questions about your variables and console session
[MELISSA]

Context refers to the information Positron Assitant provides to the LLM alongside your prompt. 

Positron Assistant provides context about files and folders, and it also provides dynamic in memory data from your session.

This includes your loaded variables, plots pane, active console session, and session state. With your permission, assistant can even run code directly in your active session.

You can take advantage of this context by asking assistant to plot data frames for you or tweak things like the title or axis labels on  your plots.

Asking about the current plot with Console session enabled in context
[MELISSA]

You can view what context Assistant is providing to the LLM in your chat context.

Here we ask Assistant about the current plot with the console session enabled as context. Assistant has access to review the code that generated the plot, as well as the plot image itself.
The open eye icon indicates that the context is visible to assistant.
